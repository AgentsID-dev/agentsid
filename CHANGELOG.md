# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-25

### Added
- Initial release
- TypeScript, Python, Ruby, Java SDKs
- CLI tool (`npx agentsid`)
- HMAC-SHA256 token signing with key rotation
- Deny-first permission engine (wildcards, conditions, schedules, rate limits)
- Delegation chains with scope narrowing
- Tamper-evident SHA-256 audit hash chain
- Approval gates for sensitive actions
- Webhook integrations (8 event types)
- Dashboard with agent management, live audit feed, agent constellation
- React frontend with docs, guides, landing page
