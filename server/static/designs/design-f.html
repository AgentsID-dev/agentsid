// how it works

Three lines. Five minutes. Done.

01

Install the SDK

One package. No dependencies. TypeScript, Python, Ruby, Java.

npm install @agentsid/sdk
02

Register your agents

Give each agent a name, scoped permissions, and a TTL. Get back a signed token.

aid.registerAgent({ name, permissions, ttl })
03

Add middleware

One line wraps your MCP server. Every tool call is verified, permissioned, and logged.

server.use(agentsIdMiddleware({ ... }))
// real code

What it looks like in your codebase

server.ts TypeScript
import { AgentsID } from '@agentsid/sdk'; import { agentsIdMiddleware } from '@agentsid/sdk/mcp'; // Initialize the client const aid = new AgentsID({ projectKey: process.env.AGENTSID_KEY }); // Register an agent with scoped permissions const { agent, token } = await aid.registerAgent({ name: 'research-assistant', onBehalfOf: 'user_abc', permissions: ['search_memories', 'save_memory'], ttlHours: 24 }); // One line of middleware. That's it. server.use(agentsIdMiddleware({ projectKey: process.env.AGENTSID_KEY, getToken: (req) => req.headers['x-agent-token'], })); // Every tool call is now: // 1. Token signature verified // 2. Permissions checked against scope // 3. Action logged to immutable audit trail
// features

Everything you need. Nothing you don't.

ID

Agent Identity

Cryptographic identity for every agent. Unique IDs, signed tokens, automatic key rotation.

PM

Scoped Permissions

Fine-grained permissions per agent, per tool, per user. Principle of least privilege enforced.

AU

Audit Trail

Immutable log of every action. Who did what, when, on whose behalf. SOC 2 and HIPAA ready.

MW

MCP Middleware

Drop-in middleware for any MCP server. One line of code. Automatic verification on every tool call.

RV

Instant Revocation

Revoke any agent instantly without affecting others. Automatic expiry with configurable TTL.

DB

Real-time Dashboard

Live view of all agents, permissions, and activity. Searchable audit logs. Alert on anomalies.
