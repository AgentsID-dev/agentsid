Welcome to AgentsID

Let's get your first agent protected in 5 minutes.

1

Copy your API key

Your project key is saved in your browser. You'll need it to connect your AI tools.

2

Install the SDK

npm install @agentsid/sdk
3

Register your first agent

Use the CLI or paste this code into your project:

import { AgentsID } from '@agentsid/sdk';

const aid = new AgentsID({ projectKey: 'YOUR_KEY' });

const { agent, token } = await aid.registerAgent({
  name: 'my-first-agent',
  onBehalfOf: 'user_123',
  permissions: ['search_*', 'save_memory'],
});
